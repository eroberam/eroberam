## How I approach AI projects

I believe that a good AI system is not defined only by a high metric on a test set.

A serious AI project requires:

- A clear definition of the problem
- A well-designed dataset and split protocol
- Strong baselines
- Controlled experiments
- Robust evaluation metrics
- Error analysis
- Explainability when decisions matter
- Awareness of bias, leakage and domain shift
- Reproducible methodology

This is especially important in biomedical, biometric and security-related applications, where an apparently good model can still be unsafe, biased or unreliable if it has not been evaluated correctly.
